TwinCAT 3 Project Creation

To create a TwinCAT 3 project:

  1. From the top menu, select the option File→New→Project.

    Creating a new TwinCAT project

  2. In the dialog box that appears, choose TwinCAT XAE Project template (1), set the project name and path (2), and press OK (3).

    Choosing TwinCAT XAE template

  3. In the project tree, right-click on the PLC node and choose Add New Item.

    Adding a new PLC project

  4. In the dialog box that appears, choose Standard PLC Project template (4), set the PLC project name (5) and click Add (6).

    Choosing a Standard PLC Project template

The above-mentioned steps result in a project tree containing folders: External Types, References, DUTs, GVLs, POUs, VISUs.
